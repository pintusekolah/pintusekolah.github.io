9. | 4 / √7+√11 | = a. √7+√3) b. √7-√11 c. √11-√7 d. -√7 e. -√11


Nilai dari [tex] | \frac{4}{ \sqrt{7} + \sqrt{11} } | [/tex] adalah [tex] \sqrt{11} - \sqrt{7} [/tex] (C).

Pembahasan

Dari segi geometris, nilai mutlak x dituliskan sebagai | x |, yaitu jarak dari x ke 0 pada garis bilangan real. Karena jarak selalu positif atau nol, untuk setiap x bilangan real, nilai mutlak x selalu positif atau nol.

Secara formal, nilai mutlak x didefinisikan dengan :

[tex] |x| = x \: \: \: \: \: \: jika \: x \geqslant 0[/tex]

[tex] |x| = - x \: \: \: \: \: jika \: x < 0[/tex]

Contohnya :

|5| = 5|10| = 10|-6| = -(-6) = 6

Definisi di atas dapat dijelaskan sebagai berikut:

Nilai mutlak dari angka positif atau nol adalah angka itu sendiri, sedangkan nilai mutlak dari angka negatif adalah kebalikan dari angka tersebut.

Langsung saja kita bahas soal diatas

[tex] | \frac{4}{ \sqrt{7} + \sqrt{11} } | [/tex]

[tex] = \frac{4}{ \sqrt{7} + \sqrt{11} } \times \frac{ \sqrt{7} - \sqrt{11} }{ \sqrt{7} - \sqrt{11} } [/tex]

[tex] = \frac{4( \sqrt{7} - \sqrt{11} ) }{7 - 11} [/tex]

[tex] = \frac{4( \sqrt{7} - \sqrt{11} }{ - 4} [/tex]

[tex] = - ( \sqrt{7} - \sqrt{11} )[/tex]

[tex] = - \sqrt{7} + \sqrt{11} [/tex]

[tex] = \sqrt{11} - \sqrt{7} [/tex]

Jadi, hasilnya adalah [tex] \sqrt{11} - \sqrt{7} [/tex]

31. Hasil dari (7² x 11³) : (3 x 11⁴ x 5²) x (9 x 7 x 5⁴) adalah.. a. 3 x 5 x 7³ per 11 b. 3 x 5 x 7³ per 11 c. 3 x 5² x 7² per 11 d. 3 x 5² x 7³ per 11


Menentukan hasil dari faktor pangkat suatu perkalian

(7² x 11³) : (3 x 11⁴ x 5²) x (9 x 7 x 5⁴) adalah.. a. 3 x 5 x 7³ per 11 b. 3 x 5 x 7³ per 11 c. 3 x 5² x 7² per 11 d. 3 x 5² x 7³ per 11

Pertama tama kelompokkan terlebih dahulu anatar yang perkalian dengan pemabgian menjadi satu, kemudian kalikan terlebih dahulu angka yang sama

Contoh

x² . x² = x⁴

Perkalian pangkat adalah penjumlah antara pangkat tersebut, sedangkan pembagian adalah pengurangan dari nilai pangkat tersebut

(7² x 11³) : (3 x 11⁴ x 5²) x (9 x 7 x 5⁴)

(7² x 11³) x (9 x 7 x 5⁴) / (3 x 11⁴ x 5²)

7² x 7 x 11³ x 9 x 5⁴ / (3 x 11⁴ x 5²)

7³ x 11³ x 9 x 5⁴ / 3 x 11⁴ x 5²

7³ x 9 x 5² / 3 x 11

7³ x 3 x 5² / 11

Maka jawabannya adalah D

Pembahasan

Sehingga |x| tidak pernah negatif.

..

Diketahui :

|x - 4| = 7

Ditanya :

Nilai x yang memenuhi ?

Jawab :

Definisi nilai mutlak

|x - 4|

x - 4, jika x ≥ 4-(x - 4), jika x < 4

..

°•° Untuk x ≥ 4

x - 4 = 7

⇔ x = 7 + 4

⇔ x = 11 [ Memenuhi ]

°•° Untuk x < 4

-(x - 4) = 7

⇔ -x + 4 = 7

⇔ -x = 7 - 4

⇔ -x = 3

⇔ x = -3 [ Memenuhi ]

..

Jadi, Himpunan Penyelesaian dari Persamaan Nilai Mutlak tersebut adalah Hp : { -3, 11 }.
