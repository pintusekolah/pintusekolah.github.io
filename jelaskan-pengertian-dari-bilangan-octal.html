2. 1. Konversi bilangan octal kebiner 6802 ⁸ =...² 2. konversi bilangan octal ke heksdesimal 2587 ⁸ =....¹⁶​


Jawaban:

1. Konversi bilangan oktal ke biner:

6802₈ = ...₂

Untuk mengkonversi bilangan oktal ke biner, kita perlu mengkonversi setiap digit oktal menjadi tiga digit biner.

Angka oktal 6802₈ terdiri dari empat digit yaitu 6, 8, 0, dan 2.

Digit oktal 6 = 110

Digit oktal 8 = 1000

Digit oktal 0 = 000

Digit oktal 2 = 010

Jadi, hasil konversi adalah 11010000010₂.

2. Konversi bilangan oktal ke heksadesimal:

2587₈ = ...₁₆

Untuk mengkonversi bilangan oktal ke heksadesimal, kita perlu mengkonversi setiap digit oktal menjadi digit heksadesimal.

Angka oktal 2587₈ terdiri dari empat digit yaitu 2, 5, 8, dan 7.

Digit oktal 2 = 2

Digit oktal 5 = 5

Digit oktal 8 = 8

Digit oktal 7 = 7

Namun, dalam sistem heksadesimal, digit terbesar yang dapat digunakan adalah 15. Jadi, kita perlu mengubah digit 8 menjadi digit heksadesimal yang setara, yaitu digit "8" menjadi "8" dan "9" menjadi "9", kemudian mengubah sisanya sesuai konversi heksadesimal.

Jadi, hasil konversi adalah 2587₈ = 215₍₁₆₎.

37. Carilah hasil konversi dari bilangan hexadecimal (1ae15) menjadi bilangan octal.


Jawaban:

1AE15₍₁₆₎ = 327025₍₈₎

Penjelasan:

Untuk konversi bilangan heksadesimal menjadi bilangan oktal, bisa menggunakan setidaknya 2 cara, yaitu:

Cara pertama: Heksadesimal → Desimal → OktalCara kedua: Heksadesimal → Biner → Oktal

Kali ini akan digunakan cara kedua.

Bilangan heksadesimal: 1AE15₍₁₆)

Konversi Heksadesimal ke Biner (4 bit)

1 = 0001

A = 1010

E = 1110

5 = 0101

Maka:

1AE15₍₁₆₎ = 00011010111000010101₍₂₎

Konversi Biner ke Oktal

Bilangan biner dikelompokkan per 3 bit dari belakang, lalu per kelompok dikonversi ke digit oktal.

Pengelompokannya:

(00) 011 010 111 000 010 101

         3    2   7    0      2    5

(Bit 0 di depan diabaikan saja)

Maka, bilangan oktalnya adalah:

327025₍₈₎

Tambahan (pemeriksaan hasil):

Konversi bilangan heksadesimal dan oktal ke desimal, dan periksa apakah hasilnya sama.

1AE15₍₁₆₎ = 5 + 1×16 + 14×16² + 10×16³ + 1×16⁴

             = 5 + 16 + 3584 + 40960 + 65536

             = 110101₍₁₀₎

327025₍₈₎ = 5 + 2×8 + 0×8² + 7×8³ + 2×8⁴ + 3×8⁵

                = 5 + 16 + 0 + 3584 + 8192 + 98304

                = 110101₍₁₀₎

Bilangan desimal hasil konversinya sama persis.
