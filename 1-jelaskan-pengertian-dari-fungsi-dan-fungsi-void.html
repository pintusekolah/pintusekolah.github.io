2. jelaskan perbedaan fungsi void main dengan class dalam java


Perbedaan fungsi void main dengan class dalam java adalah, void main adalah main function atau fungsi main, di mana program pertama kali dieksekusi. Sedangkan class adalah kerangka  dari setiap object individual yang akan dibuat. Program dalam class hanya akan dieksekusi ketika objek dari class itu diakses melalui program dalam main function.

Pembahasan

Fungsi void main adalah sebuah fungsi utama dalam program java yang akan dijalankan oleh program ketika dieksekusi. Fungsi main ini berisi alur program yang dibuat, yang isinya akan memanggil fungsi fungsi lainnya.

class adalah kerangka  dari setiap object individual yang akan dibuat. Berikut adalah contoh  implementasi class dalam pemrograman Java:

class Bicycle

{

int cadence = 0;

int speed = 0;

int gear = 1;

}

Pada contoh di atas variabel cadence, speed, gear merupakan state atau

keadaan dari object yang akan dibuat. Class juga memiliki method yang

berfungsi sebagai alat interaksi class dengan dunia di luar class.

Contoh membuat objek dari class dalam fungsi main

public static void main(String[] args)

{

    // Membuat dua object sepeda yang berbeda

    Bicycle bike1 = new Bicycle();

    Bicycle bike2 = new Bicycle();

}

Class bisa memiliki fungsi yang disebut method.Method merupakan suatu operasi berupa fungsi-fungsi yang dapat dikerjakan oleh suatu object. Method yang didefinisikan pada class bisa dipanggil melalui object.

Method memiliki 4 (empat) bagian dasar :

Nama metode Kembalian yang dikembalikan setelah memanggil method. Input parameter. Badan atau isi method, berupa instruksi untuk melakukan sesuatu hal.

16. 1)Tuliskan "urutan" pin digital pada arduino Uno 2)tuliskan "urutan" pin analog pada arduino3)tuliskan fungsi dari void setup()dari app tinkercad​


Jawaban:

1. Urutan Pin Digital pada Arduino Uno:

Arduino Uno memiliki 14 pin digital yang diberi label sebagai D0 hingga D13. Pin D0 dan D1 digunakan sebagai pin serial komunikasi UART (Universal Asynchronous Receiver Transmitter) untuk menghubungkan Arduino dengan perangkat lain, seperti komputer. Pin D2 hingga D13 dapat digunakan sebagai input atau output digital.

2. Urutan Pin Analog pada Arduino Uno:

Arduino Uno memiliki 6 pin analog yang diberi label sebagai A0 hingga A5. Pin-pin ini dapat digunakan sebagai input analog, yang memungkinkan pembacaan tegangan antara 0 hingga 5 volt. Selain itu, pin-pin analog ini juga dapat digunakan sebagai pin digital.

3. Fungsi dari void setup() pada app Tinkercad:

Fungsi dari void setup() pada app Tinkercad adalah untuk menentukan pengaturan awal dari program. Biasanya, dalam blok kode ini kita menentukan pin input/output, menginisialisasi komunikasi serial, atau menentukan kecepatan transmisi data. Blok kode dalam void setup() hanya dijalankan satu kali ketika program dijalankan pada awalnya.

4.  - Pendeklarasian Fungsi:

void sapa(nama){

   cout << "Hello, " << nama << endl;

}

void = sebuah tipe dari fungsi, lebih tepatnya fungsi tanpa kembalian.

sapa = sebagai nama fungsi yang telah kita buat dan akan digunakan untuk memanggil fungsi itu nantinya di main program.

{} = menandakan kode yang ada di dalam tanda kurung kurawal ini sebagi bagian dari milik fungsi sapa.

nama = sebagai paramater.

cout = berarti console out, atau sederhannya menampilkan hasil dari program ke console/terminal.

<< = tanda yang berati akan mengeluarkan/menampilkan ke terminal

"Hello" = teks/string biasa.

nama = akan menampilkan nama(Parameter) ke console/terminal.

endl = bisa diartikan sebagia "\n" atau disebut juga dengan new line/baris baru.

; = titik dua/semicolom berarti sebagai penanda jika kode pada baris ini akan berhenti.

- Pendefinisian(pemanggilan?) :

#include <iostream>

using namespace std;

int main(){

string nama;

cout << "masukan nama kamu : " ; cin >> nama;

sapa(nama);

}

#include <iostream> = prosesor pengarah yang mengatakan kepada kompiler untuk meletakan kode dari header file iostream.

using namespace std; =  membuat kita tidak perlu menggunakan std:: lagi di dalam program.

int main() = sebagi main program.

{} = menandakan kode yang ada di dalam tanda kurung kurawal ini sebagi bagian dari milik main program.

str nama; = sebagai variabel yang menyimpan data/value yang dimasukan oleh user.

cout << =  berarti console out, atua sederhannya menampilkan hasil dari programnya ke console/terminal.

; = berarti sebagai penanda jika kode pada baris ini akan berhenti.

cin >> = console input, yang berarti user akan memasukan/input data ke variabel nama;.

sapa(nama); = memnaggil fungsi. kita akan mengoper data yang telah di-input-kan oleh user di variabel nama; ke dalam fungsi untuk natinya ditampilkan ke dalam console/terminal.

; = berarti sebagai penanda jika kode pada baris ini akan berhenti.
