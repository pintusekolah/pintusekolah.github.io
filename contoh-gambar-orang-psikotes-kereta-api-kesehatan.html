34. diketahui sebuah kereta api digambarkan dengan skala 1:30 jika lebar kereta api sebenarnya adalah 1,2 m berapakah lebar kereta api pada peta​


Jawaban:

19.0,04 meter

Penjelasan dengan langkah-langkah:

Pertanyaan ini berkaitan dengan konsep skala dalam matematika. Skala 1:30 berarti setiap 1 unit pada gambar atau model mewakili 30 unit pada objek aslinya. Oleh karena itu, untuk mengetahui lebar mobil pada gambar atau model, kita perlu membagi lebar asli mobil dengan skala yang diberikan. Dalam hal ini, lebar asli mobil adalah 1,2 meter. Maka, 1,2 meter dibagi dengan 30 menghasilkan 0,04 meter atau 4 cm. Jadi, lebar mobil pada gambar atau model adalah 0,04 meter atau 4 cm.
